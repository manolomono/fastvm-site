Funcionalidades

Todo lo que necesitas para gestionar tus VMs

Una interfaz moderna y completa para administrar tu infraestructura de virtualizacion.

Dashboard Moderno

Interfaz dark con estadisticas en tiempo real, grid de VMs con acciones rapidas y auto-refresco cada 10 segundos.

Consola SPICE HTML5

Accede a tus VMs directamente desde el navegador via WebSocket. Clipboard bidireccional, pantalla completa y sin puertos extra.

Monitoreo en Tiempo Real

Metricas de CPU, RAM y disco con graficos Chart.js. Push via WebSocket con historial de 10 minutos por VM y host.

Gestion de Volumenes

Crea volumenes qcow2 o raw, adjunta y desconecta en caliente. Snapshots con creacion, restauracion y eliminacion.

Seguridad Integrada

Autenticacion JWT, hashing bcrypt, rate limiting, audit logging y soporte para roles de usuario (admin/regular).

Hardware Avanzado

UEFI, Secure Boot, TPM 2.0 emulado, QXL optimizado para SPICE y seleccion de modelo de CPU (host, Skylake, EPYC).

Guest Agent

Integracion con QEMU Guest Agent para info del SO en tiempo real: IP, hostname, uptime y uso de disco.

Cloud-Init

Genera ISOs cloud-init para provisionamiento automatizado. Configura hostname, usuarios y paquetes desde la interfaz.

Redes

Networking flexible y potente

Configura la red de tus VMs segun tus necesidades con multiples modos disponibles.

  • NAT — Red aislada con acceso a Internet y port forwarding
  • Bridge — Conecta la VM directamente a tu red fisica
  • MacVTAP — Acceso directo sin bridge, alto rendimiento
  • Isolated — Red completamente aislada entre VMs

Instalacion

Listo en minutos

Elige el metodo que mejor se adapte a tu entorno.

Recomendado

Instalador Interactivo

Instala dependencias, configura Docker, red y servicios automaticamente.

sudo bash install.sh

Docker Compose

Para entornos con Docker ya configurado. Un solo comando.

docker compose up -d --build

Manual

Control total sobre la instalacion. Requiere QEMU, KVM, Python 3 y dependencias.

git clone https://github.com/manolomono/fast_vm.git
cd fast_vm && pip install -r requirements.txt
./start.sh
API REST

API completa y documentada

Todos los endpoints protegidos con JWT. Gestiona VMs, volumenes, snapshots, backups y metricas de forma programatica.

POST /api/auth/login
GET /api/vms
POST /api/vms
POST /api/vms/{id}/start
GET /api/vms/{id}/snapshots
WS /ws/spice/{vm_name}
api-example.sh
# Autenticacion
$ TOKEN=$(curl -s -X POST \
  http://localhost:8000/api/auth/login \
  -d '{"username":"admin","password":"admin"}' \
  | jq -r '.access_token')

# Listar VMs
$ curl -s \
  -H "Authorization: Bearer $TOKEN" \
  http://localhost:8000/api/vms | jq
